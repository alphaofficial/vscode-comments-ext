# Support

## Issues

Report bugs and request features here:

- https://github.com/alphaofficial/vscode-comments-ext/issues

## What To Include

When reporting a problem, include:

- VS Code version
- Extension version
- Operating system
- Whether you are using local VS Code or Remote SSH
- Steps to reproduce
- Relevant output or screenshots

## Workspace Notes

Margin stores local comments in `.vscode/margin.json` and generates `.vscode/bin/*` for workspace CLI access. If something looks wrong, include whether the workspace was initialized with `Margin: Initialize Margin`.
